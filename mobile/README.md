# rice_guard

A new Flutter project.

## Run (Dev)

```bash
flutter run \
  --dart-define=ENV=dev \
  --dart-define=API_BASE_URL=http://10.0.2.2:8080
```

API calls target the versioned backend routes (`/api/v1/*`).

## Display Settings

Open the tune icon on the home app bar to change:
- Light / dark / system theme
- Font size
- Font family
